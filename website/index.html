Highly Available Web Application

Deployed using AWS EC2, ALB, S3, and VPC

Project Overview

This web application demonstrates a highly available architecture built on Amazon Web Services. The application is served through an Application Load Balancer, distributing traffic across multiple EC2 instances running Nginx.

AWS Services Used

  • VPC & Subnets – Custom networking across multiple Availability Zones
  • EC2 – Nginx web servers
  • S3 – Centralized storage for website files
  • Application Load Balancer – Traffic distribution and fault tolerance
  • IAM Roles – Secure access between services
  • CloudWatch – Monitoring and health metrics

Automation & Availability

  • Automated deployment using EC2 User Data
  • Website content pulled from Amazon S3
  • Health check configured using /health.html
  • High availability ensured through multi-AZ design
